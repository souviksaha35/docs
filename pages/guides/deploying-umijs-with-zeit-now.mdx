[UmiJS](https://umijs.org/) is a pluggable enterprise-level React application framework.

## Step 1: Set Up Your Umi Project

Set up an Umi app with [`npm`](https://www.npmjs.com/) and move into the project directory, choosing the options **App**, **TypeScript**, and no additional functionality where prompted:

<Snippet dark text="npm init umi my-umi-project && cd my-umi-project" />
<Caption>Initializing an Umi app with <Link href="https://www.npmjs.com/">npm</Link> and moving into the project directory.</Caption>

## Step 2: Deploying Your Umi App with ZEIT Now

With an Umi app set up, it is ready to deploy live with [ZEIT Now](/docs/v2).

Using [Now CLI](/download), deploy your app with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Umi app with Now CLI.</Caption>

If you want to deploy your Umi app when you push to a Git repository, you can use either [ZEIT Now for GitHub](/docs/v2/git-integrations/zeit-now-for-github), [ZEIT Now for GitLab](/docs/v2/git-integrations/zeit-now-for-gitlab), or [ZEIT Now for Bitbucket](/docs/v2/git-integrations/zeit-now-for-bitbucket) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.
