[Ember](https://emberjs.com/) is a JavaScript framework for creating ambitious web applications.

## Step 1: Set Up Your Ember Project

[Set up an Ember project](https://guides.emberjs.com/release/getting-started/quick-start/) with [`npx`](https://www.npmjs.com/package/npx) and move into the project directory:

<Snippet dark text="npx ember-cli new my-ember-project && cd my-ember-project" />
<Caption>Initializing an Ember project with <Link href="https://ember-cli.com/">ember-cli</Link> and moving into the project directory.</Caption>

## Step 2: Deploying Your Ember App with ZEIT Now

With an Ember app set up, it is ready to deploy live with [ZEIT Now](/docs/v2).

Using [Now CLI](/download), deploy your website with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Ember app with Now CLI.</Caption>

If you want to deploy your Ember project when you push to a Git repository, you can use either [ZEIT Now for GitHub](/docs/v2/git-integrations/zeit-now-for-github), [ZEIT Now for GitLab](/docs/v2/git-integrations/zeit-now-for-gitlab), or [ZEIT Now for Bitbucket](/docs/v2/git-integrations/zeit-now-for-bitbucket) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.
