[Eleventy](https://www.11ty.io/) is a simpler static site generator, created to be a JavaScript alternative to Jekyll.

## Step 1: Set Up Your Eleventy Project

Set up an Eleventy site with [`npx`](https://www.npmjs.com/package/npx) and move into the project directory:

<Snippet dark text="npx degit 11ty/eleventy-base-blog my-11ty-project && cd my-11ty-project" />
<Caption>Initializing an Eleventy site with <Link href="https://www.npmjs.com/package/npx">npx</Link> and moving into the project directory.</Caption>

## Step 2: Deploying Your Eleventy Site with ZEIT Now

With an Eleventy site set up, it is ready to deploy live with [ZEIT Now](/docs/v2).

Using [Now CLI](/download), deploy your website with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Eleventy site with Now CLI.</Caption>

If you want to deploy your Eleventy site when you push to a Git repository, you can use either [ZEIT Now for GitHub](/docs/v2/git-integrations/zeit-now-for-github), [ZEIT Now for GitLab](/docs/v2/git-integrations/zeit-now-for-gitlab), or [ZEIT Now for Bitbucket](/docs/v2/git-integrations/zeit-now-for-bitbucket) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.
